新航在2022年8月大肆调整了里程换票的停留规则，取消了付费停留，免费停留也仅能在兑换往返机票的时候享受一次不超过30天的停留，使得可玩性大大降低了。最近学么着换看樱花的机票的时候，发现日本航线兑换新航自营和星盟承运的票所需要的里程居然几乎一样，这就有点可玩的了。

# 日本航线换票所需里程

KrisFlyer新航奖励计划兑换新加坡至日本所有航点所需的里程相同，均为

|                          | 单程所需里程 |
| :-----------------------: | :----------: |
| 新航承运的经济舱（Saver） |    27,000    |
| 新航承运的商务舱（Saver） |    52,000    |
|   星空联盟承运的经济舱   |    27,000    |
|   星空联盟承运的商务舱   |    52,500    |

注：只要整个行程中含有非新航承运的航段，即属于星空联盟承运。如SIN-BKK-HND，其中SIN-BKK由新航承运，BKK-HND由泰航承运，则兑换商务舱需要52,500里程。

兑换包含从新加坡离境的航段的行程，需要以现金支付新加坡离境的税费，以实际离境日期计算，2022年11月前为SGD52.3，2022年11月起为SGD59.2，2023年4月起为SGD62.2，2024年4月起为SGD65.2。

兑换包含从日本离境的航段的行程，需要以现金支付日本离境的税费，目前东京（羽田）机场为JPY3,950，成田机场为JPY3,660，关西机场为JPY4,100，福冈机场为JPY1,980，中部机场为JPY3,970。

兑换包含转机航段的行程，可能需要以现金支付中转地中转的税费。

兑换包含非SQ/AS/AV/CM/ET/SK/UA/VA实际承运的国际航段的行程，需要支付高额的YQ/YR（燃油附加费、燃油税、航空公司附加费等），故不建议兑换包含非上述航空公司承运的国际航段的行程。

# 兑换日本航线机票时里程的价值

以下预设了三种情况，并在每种情况下寻找了近期往返日本所有航点中的最低票价，以此计算里程的价值，方便你决定究竟是选择里程卡还是返现卡。三种情况分别为：廉航或需要转机的经济舱，以中国国航为例；新航经济舱；新航商务舱。请注意，这个估计因为选择的是可以查到的最低票价，且没有考虑到里程换票的灵活性和其他换票福利，所以算是极为保守的。

| 航空公司 |   舱位   | 往返不含税票价 |    里程价值    |
| :------: | :------: | :------------: | :------------: |
|    CA    | 经济舱 L |     SGD390     | 0.72 cent/mile |
|    SQ    | 经济舱 K |     SGD600     | 1.11 cent/mile |
|    SQ    | 商务舱 D |    SGD3,088    | 2.97 cent/mile |

对于第一种情况：如果你可以接受廉航或转机，那么往返SGD400的票还是很多的。此时若是使用54,000里程兑换，每单位里程的价值仅为0.72分，4mpd的信用卡相当于2.88%的返现，并不如5%的返现卡来的划算。这种情况下，选择返现卡更加合适。

对于第二种情况：如果你认可新航/ANA经济舱的服务（反正我是不认可），或是需要节省宝贵的假期时间，那么选择直飞往返票价一般在SGD600以上。此时若是使用54,000里程兑换，每单位里程的价值为1.11分，4mpd的信用卡相当于4.44%的返现。虽然不及5%的返现卡高，但是考虑到里程卡的方便性（无低消、高封顶），以及里程换票的灵活度（旺季出行或者明天就要出行），里程卡还是值得考虑的。

对于第三种情况：虽然商务舱的标价来到了SGD3,000以上，但是真正以这个价格自掏腰包买商务舱机票的普通人应该也没有几个。商务舱的实际价值实在是非常主观，所以这个每单位里程价值为2.97分的计算更像是一个上限。这种情况下里程的价值究竟有多少，或许每个人的答案都不一样。需要注意的是，虽然换商务舱比换经济舱更值得（如果你认可商务舱的话），但是日本去程的商务舱实在是非常难换，樱花季更是连候补的机会都没有。所以有时候你不得不选择去程经济舱、回程商务舱的行程，而此时相较于兑换商务舱往返，里程的价值会被稀释。

# 现金购票与里程换票的优劣

|              现金购票              |        里程换票        |
| :--------------------------------: | :---------------------: |
| 价格随淡旺季、距出发日期的远近波动 |        价格不变        |
|              保证出行              |       旺季换票难       |
|  单程与往返票价几乎相同（除廉航）  |  单程为往返票价的一半  |
|      退改签费用高或不能退改签      | 退票/改签费用为USD75/25 |
|            不能更改行程            |   更改行程费用为USD25   |

# 里程换票的额外福利

### 追加一段日本国内航线

目的地是非新航直飞城市（东京、大阪、福冈和名古屋），又不想第一天就开始使用JR Pass？想去的日子飞东京没票了，但是飞大阪的票？这些情况或许可以通过支付一点点费用加订一段日本国内航段解决。

可追加的航线：挂ANA代码（可由其他航空公司承运）的日本国内航线经济舱均可追加。转机停留时间不可超过24小时。具体航线可至[ANA网站](https://www.ana.co.jp/en/sg/plan-book/routes-timetables/timetable/routemap/)查询。

可预定的条件：新加坡至日本的国际航段有可供兑换的由新航承运的Saver票 且 日本国内航段有可供兑换的经济舱票。二者均可在新航官网查询。

需要额外支付的里程：由于包含了ANA代码的日本国内航线，故整个行程属于星空联盟承运。新加坡至日本的国际航段为新航承运的经济舱时不需要额外支付里程，新加坡至日本的国际航段为新航承运的商务舱时需要额外支付500里程。

需要额外支付的现金：需要额外支付日本国内航段的税费，具体数额取决于日本国内航线的起降机场。具体费用可至[ANA网站](https://www.ana.co.jp/en/jp/guide/plan/charge/pfc/)查询。

换票方式：若新加坡至日本的国际航段为新航承运的经济舱、且日本国内的最终目的地不为新航直飞航点、且转机不过夜，则可通过新航官网直接兑换；否则需要通过电话兑换。

### 追加一段东南亚航线

只去日本玩不够爽？商务舱只坐一段不够爽？这时候支付（或者倒贴）一点点费用即可加订一段东南亚航线。

可追加的航线：新航承运的新加坡至马来西亚、印度尼西亚、文莱、菲律宾、泰国、越南、缅甸、柬埔寨和老挝的所有航线。新加坡转机停留时间不可超过24小时或30天（仅限兑换新航承运的往返票[^1]）。

可预定的条件：新加坡至日本的国际航段有可供兑换的由新航承运的Saver票 且 新加坡至东南亚的国际航段有可供兑换的由新航承运的Saver票。二者均可在新航官网查询。

需要额外支付的里程：不需要额外支付里程

需要额外支付（倒贴）的现金：若回程追加一段东南亚航线（即日本-新加坡-东南亚），需要额外支付新加坡中转的税费（目前为SGD9）；若去程追加一段东南亚航线（即东南亚-新加坡-日本），需要额外支付始发国/机场离境的税费，并退还新加坡离境与中转的税费之间的差额（2022年11月前为SGD43.3，2022年11月起为SGD50.2，2023年4月起为SGD53.2，2024年4月起为SGD56.2），累积起来一般会倒贴给你钱。

换票方式：新航官网直接兑换

[^1]: 往返均需由同一个非新加坡的东南亚航点出发/到达，且需全程为新航承运。往返仅有一次可在新加坡停留超过24小时。建议选择去程停留回程弃程的方式一次安排两个行程。以曼谷为例，兑换BKK-SIN-HND-SIN-BKK的行程，其中第一次SIN转机时停留超过24小时；自己另行购买/兑换一段之前的SIN-BKK；首先去泰国玩，泰国回到新加坡后在SIN停留最多30天，接着去日本玩，日本回到新加坡后弃掉最后的SIN-BKK。
