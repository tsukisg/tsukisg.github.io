# 什么是里程（信用）卡？

信用卡一般分为两种：返现卡（Cashback）和里程卡（Miles/Rewards）。返现卡，顾名思义就是刷卡消费之后，银行会回馈一定比例的返现。而里程卡则是刷卡消费之后，银行会回馈积分。银行积分可以兑换礼品，但是兑换比例往往比较坑，而用来兑换航空公司里程一般可以让其价值最大化。

航空公司里程是各个航空公司常旅客计划的积分，可以用来兑换机票，也可以用来在购买机票时抵扣现金，或者兑换礼品。当你注册成为航空公司的会员后，获取航空公司里程的主要方式除了坐飞机以外，就是通过银行的信用卡积分兑换了。

因为各个银行消费获得的积分比例不同，积分兑换航空公司里程的比例也不同，所以这篇文章中均用mpd（每消费$1可获得的航空公司里程）衡量，不会涉及银行积分的具体数量。

# 选择里程卡还是返现卡？

目前新加坡的无脑返现卡（没有复杂的消费分类，没有月最低消费的要求）的返现比例一般在2%以下，如SCB Spree（2%，限每月前\\$2,000消费）或UOB Absolute Cashback（1.7%）等。而比例比较高的返现卡，往往又有着比较严苛的条件，如Citi Cashback（8%，每月消费需为\\$800-\\$1000，且仅限超市），UOB EVOL（6.67%，每月消费需为\\$600，且线上和线下消费均不超过\\$350）等。

而里程卡往往没有最低消费要求，也没有复杂的类别要求（一般仅区分线上和线下消费），里程比例一般为4mpd。如果你能将每英里里程用到1分钱以上的价值的话（对于需要自费坐飞机的人来说并不难，每英里里程在不同使用方式下的价值请见下节），就相当于4%的返现。

不过大人的选择当然是“我全都要”。我个人的策略是优先刷满（能刷到最低消费要求的）返现大于5%的返现卡，其余的消费使用里程卡。

# 里程价值多少？

里程的具体价值取决于你如何使用。最“值得”的方式是用来兑换航空公司的机票。若只按照票价算，用于兑换头等舱/商务舱时，价值远高于用于兑换经济舱。当然，如果你是一个相比于体验商务舱，更愿意乘坐经济舱去更多的地方旅行的人，兑换经济舱机票对你来说可能更值得。

以下是以KrisFlyer新航奖励计划里程为例，各种使用方式下1英里里程的价值

| 使用方式                | 每英里里程价值 |
| :---------------------- | :------------- |
| 兑换新航航班            | 1分-6分新币    |
| 为新航航班付款时抵扣    | 0.95分新币     |
| KrisFlyer vRoom酒店预定 | 0.8分新币      |
| KrisShop购物            | 0.8分新币      |
| 转换为香格里拉会积分    | 0.77分新币     |
| 转换为凯德星积分        | 0.7分新币      |

因为凯德星积分基本可以用来做任何事情，所以每英里里程价值小于0.7分新币的使用方式没有实际意义，这里就不列出了。

# 通用里程卡分析

## 所用到的一些名词的解释

| 里程比例     | 在满足奖励条件的情况下，每消费$1所获得积分可兑换的里程<br />（其由基础积分和奖励积分组成，基础积分在不满足奖励条件的情况下亦可取得） |
| :----------- | :----------------------------------------------------------------------------------------------------------------------------------- |
| 消费单位     | 在计算单笔消费所累积的积分时，消费金额向下取整的单位[^3]                                                                             |
| 线上奖励类别 | 可获得奖励积分的线上消费所需要满足的类别[^4]                                                                                         |
| 线下奖励类别 | 可获得奖励积分的线下消费所需要满足的类别[^4]                                                                                         |
| 入账时间     | 基础积分和奖励积分的入账时间                                                                                                         |
| 每月最低消费 | 每月为获得奖励积分所需达到的最低消费                                                                                                 |
| 每月封顶消费 | 每月可获得奖励积分的最大消费金额，超出部分仅可获得基础积分                                                                           |
| 月份计算方式 | 每月最低消费和每月封顶消费中，“每月”的计算方式                                                                                     |
| 积分有效期   | 积分在银行的有效期                                                                                                                   |
| 最低兑换额   | 将银行积分兑换为航空公司里程时，最少所需兑换的里程数量                                                                               |
