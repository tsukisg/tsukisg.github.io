写完了里程卡，这篇我们就来看看返现卡。由于返现卡实在是太多了，而且有各种各样的坑，所以我们就先从最简单的无最低消费限制的卡说起。

# 什么是返现（信用）卡？

返现卡，顾名思义就是刷卡消费之后，银行会回馈一定比例的返现。

# 一句话就说完了？

看起来很简单对吧，但是银行为了~~骗傻子~~将利润最大化，往往在返现卡里放了各种各样的坑。比如华侨银行的FRANK，官网上写着大大的“最高10%返现”，然而下面用小字写了“新币消费8%返现，返现金额\\$25封顶”，再然而下面用更小的字写了“每月消费需达到\\$800”，所以实际上这张卡的有效返现比例仅有3.125%；再比如马来亚银行的Family & Friends，餐饮类别的返现比例是8%，但你需要仔细阅读条款后，才能发现这个“餐饮”是不包括快餐的~~（现实中你很难分辨一家店是否是快餐店，比如萨莉亚居然是快餐店）~~。

所以今天就先推荐几张不需要月最低消费，没有任何坑点的返现卡。需要注意**这些卡的返现比例基本都不高，只适合作为备用卡**。仅当你无卡可刷的时候，再考虑使用这些卡。

# 通用返现卡（无最低消费）分析

## 所用到的一些名词的解释

| 返现比例     | 在消费满足返现类别的情况下的返现比例                           |
| :----------- | :------------------------------------------------------------- |
| 返现类别     | 可获得返现的消费所需要满足的类别[^1]                           |
| 入账时间     | 返现的入账时间                                                 |
| 每月最低消费 | 每月为获得返现所需达到的最低消费                               |
| 每月封顶返现 | 每月可获得返现的封顶额                                         |
| 每月封顶消费 | 每月可获得返现的最大消费金额                                   |
| 月份计算方式 | 每月最低消费、每月封顶返现和每月封顶消费中，“每月”的计算方式 |
| 最低兑换额   | 将返现用于抵扣信用卡账单时，最少所需的返现数量                 |
