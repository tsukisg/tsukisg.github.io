# 通用返现卡（无类别限制）分析

## 所用到的一些名词的解释

| 返现比例     | 在消费满足返现类别的情况下的返现比例                           |
| :----------- | :------------------------------------------------------------- |
| 返现类别     | 可获得返现的消费所需要满足的类别[^1]                           |
| 入账时间     | 返现的入账时间                                                 |
| 每月最低消费 | 每月为获得返现所需达到的最低消费                               |
| 每月封顶返现 | 每月可获得返现的封顶额                                         |
| 每月封顶消费 | 每月可获得返现的最大消费金额                                   |
| 月份计算方式 | 每月最低消费、每月封顶返现和每月封顶消费中，“每月”的计算方式 |
| 最低兑换额   | 将返现用于抵扣信用卡账单时，最少所需的返现数量                 |

## UOB EVOL Card

| 官方链接     | [点击此处](https://www.uob.com.sg/personal/cards/credit-cards/rebates-cards/evol/index.page)                          |
| :----------- | :----------------------------------------------------------------------------------------------------------------- |
| 返现比例     | 6.67%                                                                                                              |
| 返现类别     | 所有（线下需手机非接触）                                                                                           |
| 入账时间     | 下个账单周期入账                                                                                                   |
| 每月最低消费 | $600                                                                                                               |
| 每月封顶返现 | $40                                                                                                                |
| 每月封顶消费 | \\$600，且线上和线下均不超过\\$350                                                                                 |
| 月份计算方式 | 账单月，按记账日（Posting Date）计算                                                                               |
| 最低兑换额   | 0                                                                                                                  |
| 备注         | 在[UOB$商户](https://www.uob.com.sg/personal/cards/cards-privileges/uob-dollar.page)消费不会获得6.67%返现，需避免使用 |

- 优点
  - 无
- 缺点
  - 在UOB$商户消费不会获得高比例返现，需要避免使用
- 点评
  - 返现比例高，没有奇奇怪怪的坑，这张卡毫无疑问是五星神卡。要达到宣传的6.67%返现，只需要刷到\\$600且线上和线下均不超过\\$350即可。硬要说缺点，就是达到最高返现比例的消费额为且仅为\\$600，少刷一分钱返现就没了，多刷一分钱实际返现比例就降了XD。

## DBS Live Fresh Card

| 官方链接     | [点击此处](https://www.dbs.com.sg/personal/cards/credit-cards/live-fresh-dbs-visa-paywave-platinum-card) |
| :----------- | :---------------------------------------------------------------------------------------------------- |
| 返现比例     | 5%                                                                                                    |
| 返现类别     | 所有（线下需非接触）                                                                                  |
| 入账时间     | 下个自然月入账                                                                                        |
| 每月最低消费 | $600                                                                                                  |
| 每月封顶返现 | $40                                                                                                   |
| 每月封顶消费 | \\$800，且线上和线下均不超过\\$400                                                                    |
| 月份计算方式 | 自然月，按交易日（Transaction Date）计算                                                              |
| 最低兑换额   | 0                                                                                                     |
| 备注         | 计算最低消费时没有标准除外交易；<br />部分绿色商家有额外5%的返现，每月$25封顶                         |

- 优点
  - 计算最低消费时没有标准除外交易
  - 根据交易日（Transaction Date）计算消费属于哪个月份，月底时不需要注意交易日和记账日之间的间隔
- 缺点
  - 无
- 点评
  - 返现比例高，没有奇奇怪怪的坑，这张卡毫无疑问也是五星神卡。要达到宣传的5%返现，只需要刷到\\$600-\\$800且线上和线下均不超过\\$400即可。绿色商家包括公交（SimplyGo），有额外的5%返现，所以这张卡刷公交的返现比例高达10%。另一个优势是计算最低消费时没有标准除外交易（即任何通过VISA进行的消费均计入最低消费），所以如果发生到月底突然发现这个月的最低消费还没刷够的情况，可以去MRT充值你的NETS FlashPay以凑够最低消费，然后用NETS FlashPay在SAM机上交个人所得税。
  - 如果是第一次申请星展银行的信用卡，使用[官网优惠码](https://www.dbs.com.sg/personal/cards/promotions/default.page)申请并达成相应要求，可以获得额外的一次性新客户返现。

## UOB ONE Card

| 官方链接     | [点击此处](https://www.uob.com.sg/onecards/uob-one-credit-card.html) |
| :----------- | :---------------------------------------------------------------- |
| 返现比例     | 3.33%                                                             |
| 返现类别     | 所有                                                              |
| 入账时间     | 每三个账单月为一季度，该季度后的下一个账单周期                    |
| 每月最低消费 | $500且消费5笔以上                                                 |
| 每月封顶返现 | $66.67                                                            |
| 每月封顶消费 | \$2,000                                                           |
| 月份计算方式 | 账单月，按记账日（Posting Date）计算                              |
| 最低兑换额   | $0，有效期2年                                                     |
| 备注         | 部分指定商家有额外6.67%的返现，每月$100封顶                       |

- 优点
  - 无
- 缺点
  - 需要季度内的每个月均达到指定消费要求，否则无返现
- 点评
  - 这张卡曾经是每月\$2,000消费返现可达5%的神卡，但从2022年7月起降到了3.33%。它是以开卡后的每三个账单月为一个季度，季度内每个账单月消费均达到\\$2,000，第四个账单月返现\\$200；季度内每个月消费均达到\\$1,000，第四个账单月返现\\$100；季度内每个月消费均达到\\$500，第四个账单月返现\\$50。指定商家包括牛奶公司、Grab、公交（SimplyGo）、Shopee，有额外的5%-6.67%返现。如果在指定商家消费很多的话，这张卡依然值得考虑，否则建议直接剪掉。

## HSBC Advance Credit Card

| 官方链接     | [点击此处](https://www.hsbc.com.sg/credit-cards/products/advance/)           |
| :----------- | :------------------------------------------------------------------------ |
| 返现比例     | 2.5%                                                                      |
| 返现类别     | 所有                                                                      |
| 入账时间     | 下个自然月入账                                                            |
| 每月最低消费 | $2,000                                                                    |
| 每月封顶返现 | $70                                                                       |
| 每月封顶消费 | $2,800                                                                    |
| 月份计算方式 | 自然月，按记账日（Posting Date）计算                                      |
| 最低兑换额   | $0                                                                        |
| 备注         | 持有汇丰银行EGA账户且每月新存入\\$2,000，可获得额外1%返现，每月\\$300封顶 |

- 优点
  - 无
- 缺点
  - 无
- 点评
  - 这张卡是1.5%的无脑返现卡，但是每月刷到$2,000之后有额外1%的返现，使返现比例来到了2.5%。虽然配合EGA账户可以达到3.5%的返现，但是EGA账户更好的用处是做musical chair[^2]。既然有每月$2,000以上的消费能力，考虑里程卡似乎更加合适一些。
  - 如果是第一次申请汇丰银行的信用卡，通过[官网](https://www.hsbc.com.sg/credit-cards/partners/affiliates/)或[SingSaver](https://www.singsaver.com.sg/providers/hsbc/credit-card)申请并达成相应要求，可以获得额外的一次性新客户返现。

[^1]: 即使奖励条件为“所有”，部分标准除外交易（如充值预付卡、Grab钱包，缴纳学费、公立医院费用等）也不会获得基础积分和奖励积分，详见各个银行的条款
    
[^2]: 某些银行账户如汇丰银行EGA，渣打银行e$aver等，对每月新存入的资金（即本月日均余额相较上月日均余额增加的部分）发放高额的利率，如2022年11月的EGA对新存入资金发放的利率高达5.1%，故可以通过奇数月将所有资金转到账户A，偶数月将所有资金转到账户B的方式，实现所有资金在每个月都为新存入资金，享受高额利率
